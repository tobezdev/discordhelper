# Discord Helper
A simple Discord user-app that aims to provide helpful commands to improve your experience. By tobezdev and Zffu.

## Hosting
You'll need to self-host this bot to use it. 

To install requirements:
```bash
python3 -m pip install -r .\requirements.txt
```

To run the bot:
```bash
python3 -u .\main.py
```

## Customisation & Contributing
To add custom features to your bot, simply add more to the `cogs/` directory - this is where the bot loads it's extensions. For info on how to get started, see the [Pycord v2.8.0 Docs](https://docs.pycord.dev/en/v2.8.0/) (this is the library we use).
